# Simulado de Terça - Análise e Modelagem de Sistemas

Site estático para estudar Engenharia de Requisitos e Orientação a Objetos.

## Recursos

- Simulado interativo com questões de múltipla escolha
- Modo claro e escuro
- Filtro por tema
- Cards de respondidas, acertos, erros e aproveitamento
- Resumo explicado por matéria
- Revisão automática das questões erradas
- Funciona direto no GitHub Pages, sem build

## Publicação

Arquivos preparados para publicação na raiz de um repositório GitHub Pages.
